# 투명 배경 이미지 자동 크롭

투명 PNG를 올리면 그림 끝에 여백 없이 딱 맞게 잘라주는 웹 도구입니다. 미리캔버스 업로드용으로 바로 사용할 수 있습니다.

## 기능
- 투명 PNG/WEBP의 불투명 픽셀 경계를 감지해 4면 여백 없이 크롭
- 여러 장 동시 처리 · 전체 다운로드
- 투명도 기준 / 여백(px) 조절
- 원본 / 크롭 후 크기 표시, 클릭 시 빨간 외곽선 확대 팝업으로 밀착 확인

## 사용
`index.html`을 브라우저에서 열기만 하면 됩니다. 설치 불필요.

## 배포 (GitHub Pages)
저장소에 올린 뒤 Settings → Pages → Branch를 `main`으로 설정하면 공개 URL이 생성됩니다.
